## 주요 특징

- **`useSAUserBundle` Hook**:
  - 사용자 데이터 가져오기와 상태 업데이트를 관리하는 데 중요한 역할을 합니다.
  - 효율적인 데이터 패칭을 위해 `react-query`를 활용합니다.
  - Zustand와 통합하여 상태를 관리합니다.
  - User ID, Rank, Tier, Recent Info 등 특정 사용자 관련 정보를 가져오는 여러 훅으로 구성되어 있습니다.

## 내가 배운 것

1. **훅 사용법**: `useSAUserBundle`을 개발하면서 복잡한 데이터 가져오기 프로세스를 위한 커스텀 훅을 구조화하고, 로딩 및 에러 상태를 효율적으로 처리하며, 성능 최적화를 위한 결과를 메모이제이션하는 방법을 배웠습니다.

2. **Zustand를 이용한 상태 관리**:
   - Zustand의 직관적인 API로 인해 간소화된 상태 관리를 경험했습니다.
   - 컴포넌트 트리 외부에서 상태를 관리하여 더 깔끔하고 조직적인 코드를 유지하는 방법을 배웠습니다.

3. **성능 최적화**:
   - 비동기 데이터 패칭 및 캐싱을 위한 `react-query` 활용으로 데이터 처리 성능이 크게 향상되었습니다.
   - 데이터 종속성에 기반한 컴포넌트 렌더링을 최적화하기 위해 `useMemo`를 사용했습니다.

4. **에러 처리**: `react-error-boundary`를 통합하여 애플리케이션 전반의 에러를 관리하며, 대체 UI를 제공해 사용자 경험을 향상시켰습니다.
